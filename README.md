# مستشار برنامج القطاع الخاص
### أمانة منطقة الرياض

## النشر على Vercel

1. افتح [vercel.com](https://vercel.com) وسجل دخول بحساب GitHub
2. اضغط **New Project**
3. اختر هذا الـ Repository
4. في **Environment Variables** أضف:
   - `ANTHROPIC_API_KEY` = مفتاح الـ API الخاص بك
5. اضغط **Deploy**

## الميزات
- بحث ذكي في 429 مدخل من قاعدة بيانات المشروع
- لا يهنق - يرسل فقط البيانات ذات الصلة
- واجهة عربية RTL احترافية
- شعار أمانة الرياض
- أسئلة سريعة
- محادثات محفوظة
